## Quick Start

Include the files

```html
<link rel="stylesheet" href="hint.css">
<script src="hint.js"></script>
```

Distiguish the Hints

```html
<div class="your_class" data-hint="text can contains html">
```

Initialize Hint.js

```js
document.getElementsByClassName('your_class').hint(options);
```
